# Agentia Labs Website

Static public website for Agentia Labs.

## Local preview

Open `index.html` directly or serve this folder with any static server:

```bash
python3 -m http.server 4173
```

## Publishing

This folder is intended to be published as the root of a public GitHub Pages repository under `yusentic`.

The site is Spanish-first and includes an English toggle. It is intentionally static: no build step, no package manager and no server runtime.
